# Content drafts

Candidate decision prose and batch plans live here during Phase 2B production.

**Rules:**

- Never place draft cards in `data/decisions/` until they pass review, validation, and simulation.
- Each sub-batch should include a short plan file (IDs, speaker, category, trade-off) before prose.
- After integration, move or archive drafts and update `data/content_manifest.json` via the manifest builder.

See [MINISTAN_CONTENT_STYLE_GUIDE.md](../MINISTAN_CONTENT_STYLE_GUIDE.md) for the seven-step batch workflow.
