# Tiny Dictator — Game Design

## One-sentence pitch
A funny mobile strategy game where you rule a tiny fictional country by approving absurd laws and watching your citizens, economy, army, and elite react.

## Core loop
- Start a run in Ministan.
- Each day, an advisor proposes a decision.
- The player chooses left or right.
- Resources change.
- Laws and flags are added.
- The city visually changes.
- Follow-up events appear based on prior choices.
- The run ends with a newspaper summary and a collectible ending.

## Main resources
- treasury: money and economy
- happiness: public mood
- order: stability and control
- elite_loyalty: support from powerful people

If any resource reaches 0, the run usually ends.

## MVP target
Build one playable vertical slice:
- 1 country: Ministan
- 4 resources
- 4 advisors
- 30 decisions
- 8 laws
- 5 endings
- simple city visual states
- end-of-run newspaper
